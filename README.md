# LISTEN ARTEFAKT - Заметочница с ИИ

## Описание

Веб-приложение для создания и управления заметками с интеграцией ИИ для исправления текста. Приложение работает полностью в браузере, данные хранятся в localStorage.

## Структура проекта

```
Listen/
├── index.html      # Основной файл (HTML + CSS + JS)
├── package.json    # Node.js конфигурация
├── start.bat       # Скрипт запуска (http-server)
├── favicon.svg     # Иконка
└── README.md       # Документация
```

## Запуск

```bash
npm install
npm start
```

Открыть: http://localhost:3000

## Функции

### 1. Создание заметок
- Текстовый ввод в главном поле
- Голосовой ввод через Web Speech API (кнопка микрофона)
- Добавление фото (кнопка камеры)
- Дополнительный промпт для ИИ (опционально)

### 2. ИИ-обработка (OpenRouter API)
- **Основное окно**: автоматическое исправление орфографии, пунктуации и грамматики
- **Окно промпта**: полная обработка текста по запросу пользователя
- Модель: DeepSeek через OpenRouter
- API-ключ сохраняется в localStorage

### 3. Поиск
- Фильтрация заметок по тексту в реальном времени
- Поле поиска вверху раздела заметок

### 4. Теги/метки
- Добавление тегов при редактировании заметки (через запятую)
- Удаление тегов кликом по ×
- Панель тегов для фильтрации заметок по категориям
- Кнопка "Все" сбрасывает фильтр

### 5. Закрепление
- Кнопка 📌 (pin) слева в заметке
- Закреплённые заметки показываются сверху
- Визуально выделяются рамкой accent цвета

### 6. Редактирование
- Открытие модального окна редактирования
- Редактирование текста
- Управление тегами
- Сохранение изменений

### 7. Перетаскивание (drag & drop)
- Заметки можно перетаскивать для изменения порядка
- Визуальный эффект при перетаскивании

### 8. История (вкладка "Чат")
- Просмотр всех заметок в хронологическом порядке
- Без кнопок действий (только чтение)

### 9. Настройки (вкладка "Настройки")
- Ввод и сохранение API-ключа OpenRouter
- Ключ хранится в localStorage
- By Векшин Алексей

## Структура данных заметки

```javascript
{
  id: number,           // Unix timestamp
  text: string,         // Текст заметки
  photo: string|null,   // Base64 фото
  created: number,      // Дата создания
  improved: boolean,    // Была ли обработка ИИ
  customPrompt: string, // Промпт пользователя
  tags: string[],       // Массив тегов
  pinned: boolean       // Закреплена ли
}
```

## localStorage

- `listenNotes` - массив заметок (JSON)
- `listenApiKey` - API-ключ OpenRouter

## Зависимости

- http-server (dev) - локальный сервер для разработки

## Технологии

- Чистый HTML/CSS/JavaScript
- Web Speech API (голосовой ввод)
- OpenRouter API (ИИ-обработка)
- localStorage (хранение данных)
- Fetch API (HTTP-запросы)
